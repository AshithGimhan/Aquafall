# Aquafall

A static informational website showcasing a sea turtle hatchery and rescue center project. The site is built using HTML, CSS, and JavaScript, and includes pages for categories, hatchery details, threats, donations, ticket booking, and more.

## Project Structure

- `index.html` - Home page with hero video, welcome content, mission, and contact details.
- `Turtle2.html` - Categories page.
- `Turtle3.html` - Hikkaduwa Turtle Hatchery page.
- `Turtle4.html` - Threats page.
- `Turtle5.html` - Kosgoda Hatchery page.
- `Turtle6.html` - Bentota Hatchery page.
- `Turtle7.html` - Donate page.
- `Tickets.html` - Tickets page for visitor reservations.
- `Tickets-cont.html` - Ticket details or continuation page.
- `checkout.html` - Booking checkout page.
- `confirmation.html` - Confirmation page after booking.
- `Details.html` - Additional details page.

## Assets

- `images/` - Image assets used throughout the website.
- `Video/` - Background video assets used on the homepage.

## How to use

1. Open `index.html` in a browser to view the homepage.
2. Use the navigation menu to move between pages.
3. For local testing, open any page directly or serve the folder with a local web server.

## Notes

- This is a front-end only project with no backend server.
- Ensure the `images` and `Video` folders remain in the same directory as the HTML files.
- The site is designed to highlight sea turtle conservation, hatchery education, and visitor information.
